## Required Sound Files

Place the following sound files in this directory:

1. **cursor-hover.mp3** - Light ping sound (0.3-0.5s)
2. **button-click.mp3** - Neon tap sound (0.2-0.4s)
3. **project-hover.mp3** - Sweep sound (0.5-1s)
4. **section-load.mp3** - Soft ambient swell (1-2s)
5. **globe-rotation.mp3** - Hologram hum (0.5-1s)
6. **cinematic-intro.mp3** - Deep bass + airy pad (3-5s)

## Recommended Sources

- **Freesound.org** - Free sound effects (CC0 license)
- **Zapsplat.com** - Free with attribution
- **Adobe Stock** - Premium sounds
- **Web Audio API** - Code will generate fallback tones if files missing

## File Specifications

- **Format**: MP3 (preferred) or OGG
- **Bitrate**: 128-192 kbps
- **Duration**: 0.2-5 seconds
- **Volume**: Normalized to -12dB
- **Sample Rate**: 44.1kHz

## Note

If sound files are not provided, the system will use Web Audio API to generate simple tones as fallback. The site will work without sound files, but adding them enhances the experience.
